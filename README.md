# 🧮 Calculadora em Java (Console)

Uma simples calculadora desenvolvida em Java que roda no terminal.  
Permite realizar as quatro operações básicas: **adição, subtração, multiplicação e divisão**.

---

## 🔧 Funcionalidades

- ✅ Adição de dois números inteiros  
- ✅ Subtração de dois números inteiros  
- ✅ Multiplicação de dois números inteiros  
- ✅ Divisão de dois números inteiros (com verificação de divisão por zero)  
- ✅ Interface via linha de comando (CLI)  
- ✅ Loop contínuo até o usuário decidir sair

---



## ▶️ Como executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   
2. Compile o código:   
  ```bash
  javac Calc/Main.java Calc/Calculadora.java
  ```

3. Execute:
   ```bash
     java Calc.Main
   ```
   

   ## 💡 Exemplo de uso:
   ```bash
   ===== Calculadora =====
    Escolha a operação a ser realizada:
    1 - Adição (+)
    2 - Subtração (-)
    3 - Multiplicação (x)
    4 - Divisão (/)
    5 - Sair
    Opção: 1
    Digite o primeiro número: 5
    Digite o segundo número: 3
    Resultado: 8
   ```
